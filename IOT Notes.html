CMPG-323-Project-3

This repository will be used for project 3 known as the Web App Project with Testing Patterns.
In the modern age having an application for your business is crucial. A web app is the best way
of doing things as it allows for any device with a web browser to have the app accessible. In this
Project we will be learn about the fundamentals of web apps.

Due Date

This project will end on 2022-09-29

There will be 3 Branches:

  • Main: The active branch which holds the most stable version
  • Hotfix: This branch is there for small and quick bug patches
  • Controller Async: A branch where the Controllers have their methods reworked to work with Async
  • TierTwoAsync: A branch where a Tier Two Implementation of Async was Attempted (End of Life)
  • Main Development: Main branch to stage Merges (End of Life)
  • Tier Two Given: Base Implementation of a tier two Design with only the given code

Architectural pattern: MVC (Model View Controller)

Tier System:

Tier 1: Beginner (Minimum Requirement)

Repository Classes

Tier 2: Advanced (Target Tier)

Repository Classes Repository Interfaces

Tier 3: Expert (Dream Tier)

Repository Classes Repository Interfaces Separate Projects
